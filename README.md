# Open Machine Cloud - Landing Page Specification

## Expanded Prompt

### Project Context
Open Machine Cloud is an open platform for CNC part machining collaboration. The landing page must convey industrial strength, precision engineering, and the power of a global manufacturing network.

### Hero Section (Video-First)
- **Video**: Cinematic CNC machining footage - sparks flying, precision cutting, metal being shaped
- **Headline**: Bold, impactful typography dominating the viewport
- **Subheadline**: Value proposition text
- **CTA**: High-contrast "Get Started" button

### Typography
- **Headlines**: Extra bold, condensed industrial sans-serif
- **Body**: Clean, highly readable sans-serif
- **Accent**: Monospace for technical elements

### Layout
- Full-bleed hero section (100vh)
- Asymmetric grids for visual interest
- Generous whitespace balanced with dense information blocks

### Motion
- Smooth scroll animations
- Subtle parallax effects
- Hover states with metallic sheen transitions
- Counter animations for statistics

---

## Tech Strategy

### Stack
- Pure HTML5/CSS3/JavaScript
- CSS Custom Properties for theming
- Intersection Observer for scroll animations
- CSS Grid and Flexbox for layout

### Asset Protocol
- Hero video: MP4, muted autoplay, loop
- Images: WebP/PNG with lazy loading
- Icons: Inline SVG

---

## Design System

### Color Palette (NO BLUE/PURPLE - Industrial Focus)
- **Primary Dark**: #0A0A0A (Near black)
- **Secondary Dark**: #1A1A1A (Charcoal)
- **Tertiary Dark**: #2A2A2A (Dark gray)
- **Accent Primary**: #E85C0D (Industrial orange/copper)
- **Accent Secondary**: #C4A77D (Metallic brass)
- **Neutral Light**: #F5F5F5 (Off-white)
- **Steel Gray**: #6B7280 (Machine steel)
- **Chrome**: #9CA3AF (Metallic highlight)

### Typography Scale
- **Display**: 80px/96px (Hero headline)
- **H1**: 48px/56px
- **H2**: 36px/44px
- **H3**: 24px/32px
- **Body**: 16px/24px
- **Small**: 14px/20px

### Layout Grid
- Max container: 1400px
- Column gap: 32px
- Section padding: 120px vertical
- Mobile breakpoint: 768px

---

## Sections

1. **Hero**: Video background, massive headline, CTA
2. **Value Props**: 3-column grid with icons
3. **How It Works**: Step-by-step visual flow
4. **Features**: Showcase with imagery
5. **Statistics**: Animated counters
6. **Final CTA**: Dark section with signup
7. **Footer**: Navigation and social links
